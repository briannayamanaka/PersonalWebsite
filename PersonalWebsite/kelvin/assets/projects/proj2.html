Birch Aquarium Project


The most interesting aspect of this project was that we were the first team to do any implementation. I joined the Software team of the Birch Buddies in the second quarter that it existed. Its first quarter, it was in its ideation phase. The team before us made a lot of sketches, diagrams, and flow charts of what the project's goals were. It was very challenging starting something so big from scratch. Originally, we were going to build our own server using a Raspberry Pi and host the web application ourselves. After much research, I suggested to the team that we utilize an all-in-one tool such as Firebase to handle the database, hosting, and authentication. After a few days we had a skeleton of all three components: 1) Employee Portal, 2) Visitor Registration and 3) Exhibit Kiosks. It will be years before all the exhibits at the Birch Aquarium are completely digitized, but we set a strong foundation that many others have already had the opportunity to build on.

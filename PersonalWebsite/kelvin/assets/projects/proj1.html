U.S. Presidents Project


We decided to create an educational website because I personally take a lot of interest in educating children and providing them with new, fun and interesting ways to learn. The project was completed in iterations. The first 3 iterations contained basic components of the web application, using only HTML and CSS. Later iterations contained bootstrap for the layout and JavaScript for the "favoriting" functionality. Implementing the image gallery and timeline using only CSS and HTML was not an easy task because of how interactive the two components are. The most time consuming task I had was reading and learning about every U.S. president so that I could present valuable, easy-to-understand content for each of the 43 presidents.
